En todos las paginas .html agregué el SEO con las siguientes caractaristicas:
-Descripción: mediante la etiqueta meta name description, agregué un breve resumen del contenido de las mismas.
-Palabras clave: mediante la etiqueta meta name keywords agregué las palabras claves que estarán en todas las paginas del sitio.
-Encabezado: Solo en index.html utilizo para el encabezado h1 con el titulo "Jason Ross". Las paginas internas tienen diferentes encabezados utilizando h3, h4 para los titulos de las mismas.
